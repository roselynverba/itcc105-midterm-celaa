# ASSCAT Smart Campus Assistant

## Problem Statement
The existing ASSCAT student portal is difficult to use because students must go through multiple pages and menus just to access simple information. This slows down important processes like checking grades, viewing schedules, and clearance verification. Many students experience confusion and inefficiency when using the system.

## Industry 5.0 Justification
This project applies Industry 5.0 principles by enhancing human experience through AI assistance. The system does not replace human services but supports students by providing quick responses and guidance. It improves accessibility and reduces stress while still allowing human interaction when necessary.

## Agile User Stories

1. As a student, I want a smart assistant that can guide me through campus services, so that I can easily access information without confusion.
2. As a student, I want to ask questions using natural language, so that I can interact with the system more comfortably.
3. As a student, I want the system to respond even with limited connectivity, so that I can still receive important updates.
